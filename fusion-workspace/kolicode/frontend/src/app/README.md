# App

Bootstrap de la app web minima de P10pro.

## Rol

- `createApp.ts`: monta la aplicacion en `#app` y conecta estado, tema e iconos.
- `../editor/`: render del shell del editor local.
- `../shared/`: contrato tipado del proyecto y utilidades comunes.

La carpeta ya no es una reserva vacia; forma parte del runtime ejecutable de la raiz.
