# P10pro - Creative Surface

Esta guia describe la superficie creativa real del proyecto y no una arquitectura full-stack aspiracional.

## Estado actual

- La app web minima vive en la raiz y se ejecuta con `npm`.
- `docs/` contiene vistas y notas alineadas con `config/project-context.json`.
- `creative/processing/` conserva un sketch local independiente para exploracion visual.

## Lo creativo hoy

- Editor shell local con assets reutilizables.
- Iconos SVG y helper `MotorIcon` compartidos entre app y documentacion.
- Tokens de diseno base para mantener el mismo lenguaje visual.

## No es estado actual

- `py5`, backend, IA local o render 3D.
- CRUD, auth o APIs HTTP.
- Cualquier narrativa heredada de plantilla que no este marcada como `planned` o `template_only`.
