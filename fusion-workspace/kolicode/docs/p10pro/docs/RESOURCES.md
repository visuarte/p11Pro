# Resources

## Recursos reales del proyecto

- Contrato estructurado: `config/project-context.json`
- Resumen de arranque: `README.md`
- Portal documental: `docs/index.html`
- Vista tecnica derivada: `docs/inicio/codedesign_technical_docs_v2.html`
- App web minima: `index.html` con `npm run dev`
- Prototipo Processing: `creative/processing/Sketch.pde`

## Politica de recursos

- Si un recurso no existe en disco, no debe listarse como disponible.
- Si un recurso describe una idea futura, debe figurar como `planned`.
- Si un recurso viene de plantilla y aun no aplica, debe figurar como `template_only`.
