Estructura del proyecto

▾

Opción 1 — Separación clásica

backend/
      build.gradle.kts
      src/main/kotlin/ (ChecklistApi.kt, ChecklistEstado.kt)
    frontend/
      package.json
      src/main.js
      styles/app.css
    docs/ (guia.html, WEBGL_MIGRATION_GUIDE.md, README_NEXT_LEVEL.md)

Opción 2 — Monorepo

apps/
      backend/
      frontend/
    packages/ (librerías compartidas)
    infra/ (CI/CD, scripts)
Elige Opción 1 si el equipo es pequeño. Opción 2 para monorepo con código compartido.
